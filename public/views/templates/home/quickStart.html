Step 1: Adding cards to the game board

Locate the Card Portal and start typing the name of a card. Suggestions will start to populate in a drop-down and can be selected from there. Cards will always appear under the Card Portal when making a Board State. The Card Portal is draggable. Try creating "Xenagos, God of Revels".

Cards are draggable as well and can be manipulated in several other ways:

  • Right click will bring up a context menu with many options depending on a card's type. Try to "Inspect" the card.

Step 2: Setting additional game information

Additional information about the Board State is displayed on the left side of the screen.
  • Right/left click life totals to increase/decrease them.
  • Each 'zone' has a panel that opens where you can add cards to that zone. (Available when making a Board State)
  • The Stack is a zone where cards can be added to show what spells are currently resolving. (Sorry, only spells currently)

Step 3: Saving and Navigating Your States

Once your state is finished, the toolbar on the right side of the screen will let you save and finish.
  • - The floppy disk will save the current state you are working on.
  • - Use these arrows to navigate between the states you have created. The current state is outlined between the arrows, along with the previous and next state. A "-" indicates no previous state and "N" indicates a new state that has not yet been created.
  • - Once you have finished working on each state, you can upload your Board State. After entering in some information about the Board State a link will be provided to share with others.
Other Tools:
  • - Return to Board State's homepage.
  • - Clear the game board. This will remove every card that has been created in the current state and cannot be undone.
  • - The zoom in and zoom out buttons will give you a larger or smaller view of the game board. The current zoom level is displayed in between.
  • - This button will open a panel where you can write a description for the current state (Not available in quick start). Each state will have a separate description available. When a Board State is shared and a state is loaded where a description has been written, the panel will automatically open. Clicking it a second time will close the panel.
